# Frontend

This folder contains the React + Vite application for Aegis Command.

## Purpose
The frontend provides the user interface, dashboard views, and interactive map for emergency dispatching and route optimization.

## Run locally
From the `frontend` folder:
```bash
npm install
npm run dev
```

## Scripts
- `npm run dev` — start the Vite development server
- `npm run build` — create a production build
- `npm run preview` — preview the built app locally

## Main files
- `src/main.jsx` — application bootstrap file
- `src/App.jsx` — main app wrapper
- `src/components/` — UI pages and dashboard components
- `src/index.css` — global styling and Tailwind definitions
- `vite.config.js` — Vite configuration
